## [v0.1.0] - 2025-12-01

### Added

- Initial public release of aeroapi-go
- Generated client from FlightAware AeroAPI OpenAPI specification v4.28.0
- Full API coverage for all AeroAPI endpoints
- `ClientWithResponses` wrapper for typed response handling
